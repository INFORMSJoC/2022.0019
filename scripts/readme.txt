                             For Solving RAP Instances

Under a Windows OS system, please run the executable program 'HER.exe' with the following five parameters
InstanceName is the name of each instance, e.g., Bovine.txt
The dataset is the type of tested instance, e.g., real-world
TimeLimit is the time limit (in seconds) for each run of the program, e.g., 3600 seconds
NbrRepats is the number of times running the program, e.g., 20
————————————————————————————————————————
./HER.exe  InstanceName  Dataset  TimeLimit  NbrRepeats
————————————————————————————————————————

Under a Windows OS system, please run the executable program 'ILS.exe' with the following five parameters
